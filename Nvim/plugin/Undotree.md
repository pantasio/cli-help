# Undotree
## Graph your Vim undo tree in style.
More info [UndoTree]{https://github.com/mbbill/undotree}

<F5>            :UndotreeToggle
<leader>u       :UndotreeFocus

s           saved changes
shift s     last saved change

u           undo
ctrl r      redo
shift j     previous state
shift k     next state
